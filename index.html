How It Works?

User uploads a resume: The user uploads a resume file in PDF format.

Resume screening: The system extracts the text content from the uploaded resume using PDF text extraction techniques.

NLP-based prediction: The system uses machine learning models and NLP techniques to analyze the text content and predict the job category.

Display prediction: The predicted job category is displayed to the user on the same page.

Technology Used

Flask: Python web framework for building the backend server.

JavaScript: Used for handling asynchronous form submission and updating page content dynamically.

HTML/CSS: Frontend markup and styling.

Natural Language Processing (NLP): Techniques for analyzing text data.

Machine Learning: Trained models for predicting job categories.

About

The Job Profiling System is a web application built using Natural Language Processing (NLP) techniques to analyze resumes and predict the job category based on the content of the resume.

This system aims to assist recruiters and HR professionals in efficiently categorizing resumes and identifying suitable candidates for job roles.
